# ESP32 Weather Station

A compact 2-layer weather monitoring system based on the ESP32, designed to measure and display environmental conditions.

## Features

- ESP32 microcontroller
- BME680 environmental sensor
- DHT11 temperature and humidity sensor
- OLED display interface
- Battery-powered operation
- USB programming and power input
- On-board voltage regulation

## Measurements

The system is capable of monitoring:

- Temperature
- Relative humidity
- Atmospheric pressure
- Air quality (VOC/gas resistance)
- Estimated altitude

## Design Information

- PCB Layers: 2
- Software: KiCad
- MCU: ESP32
